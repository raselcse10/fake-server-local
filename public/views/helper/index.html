What is it?

fake server local is a REST api server which return random lorem-ipsum data based on user schema. It might be helpfull for Front-end/App developers, who don't have sufficient data (might be REST api is not ready or not enough data in databse) to test their project. All you need, just create an api endpoint and output schema. Setup

First clone project from github , run npm install (issume you already have installed node.js in your computer) to install dependency. Now go to http://localhost:9920/dashboard. You should see the following image.
Initially you have to create a project. Why? Sometimes you have to create same api endpoint for multiple projects /login, /signup etc. and you are expecting different response data based on project. You can get different response data for same api endpont based on your project.
